
# Multi-Dorker
Multi thread google dorker written in python

![](header.png)

## Command to run Dorker on Termux 


```
git clone https://github.com/indiancybertroops/Multi-Dorker/
```
```
cd Multi-Dorker
```
```
chmod +x *
```
```
sh setup.sh
```

```
apt-get install python2
```

```
python2 ICT.py
```


## On linux :

```
git clone https://github.com/indiancybertroops/Multi-Dorker/
```

```

```
